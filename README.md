# Sistema de Compras - Node.js

Sistema de gerenciamento de compras desenvolvido com **Node.js, Express e EJS**.

## Tecnologias utilizadas

- Node.js
- Express
- EJS
- CSS

## Funcionalidades

- Cadastro de produtos
- Cadastro de fornecedores
- Requisições de compra
- Ordens de compra

## Como executar

```bash
git clone https://github.com/Josevoss7/SistemasCompras.Nodejs.git
cd SistemasCompras.Nodejs
npm install
npm run dev
```

Acesse:

```
http://localhost:3000
```
